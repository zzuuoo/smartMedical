# smartMedical
安卓端预约挂号系统

采用mvc开发模式开发项目
网络请求采用volley框架


### 开发安排：一天或两天一个小功能
#### 医生：查看本人值班安排(finish)，病历管理【查看病历，增加病历，删除病历，更新病历】(doing)，查看自己任务量(finish)
#### 患者：选择科室、医生进行挂号(finish)，查看病历，查看挂号(finish)，取消挂号()
#### 管理员：添加医生(finish)，删除医生(finish)，更新医生（），值班管理（界面大调整4.3）：：：添加值班(finish)，删除值班，更新值班，查看值班
